h2 Requirements - make sure you have already
    * Nodejs
    * Mongodb
    * Bower
h2 Installation
    * `npm install`
    * `bower install`
h2 Lunch Server
    * `mongod`
    * `npm install nodemon -g` - This package will reload server when we make changes to files.
